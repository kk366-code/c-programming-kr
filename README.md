# 📘 K&R (プログラミング言語C) 学習リポジトリ

このリポジトリは、デニス・リッチーとブライアン・カーニハンによる書籍『**プログラミング言語C (The C Programming Language, 2nd Edition)**』の学習記録と演習問題の解答コードを管理するために作成されました。

C言語の基礎と、効率的なプログラミングスキルを習得することを目標としています。

---

## 🎯 学習の目的

* C言語の基本文法と概念（ポインタ、構造体、入出力など）の理解。
* K&Rの演習問題に対する解答の作成と、標準的なコーディングスタイルの習得。
* GitとGitHubを用いた、実用的な開発ワークフローの確立。

---

## 📂 ディレクトリ構成

コードは書籍の章立てに対応するように整理されています。

```
c-programming-kr/
├── chapter01/  \# 第1章: やさしい入門
├── chapter02/  \# 第2章: データ型・演算子・式
├── chapter03/  \# 第3章: 制御の流れ
├── chapter04/  \# 第4章: 関数とプログラム構造
├── ...
└── chapter08/  \# 第8章: UNIXシステム・インターフェース

```

### 命名規則

| 種類 | 命名規則 | 例 | 説明 |
| :--- | :--- | :--- | :--- |
| **演習問題** (単一コード) | **`exercise-A-B.c`** | `exercise-1-5.c` | 第A章の演習B問目の解答。 |
| **演習問題** (複数コード) | **`exercise-A-B-N.c`** | `exercise-4-1-1.c` | 第A章の演習B問目に対する、N番目の異なるアプローチやバリエーション。 |
| **書籍内サンプルコード** | **`example-A-B_topic.c`** | `example-4-2_function-scope.c` | 第A章のB節にある重要なサンプルコード。 |
| **共通ファイル** | **`utils.h`** など | `kr-utils.h` | 複数の演習で共通して使用する関数や定義を格納。 |


---

## 🔧 環境とコンパイル方法

* **OS**: macOS / Linux / Windows (WSL)
* **コンパイラ**: GCC (GNU Compiler Collection)
* **標準**: C99 または C11

### コンパイル例

特定の演習ファイルをコンパイルする際は、以下のコマンドを使用します。

```bash
# 例: 第1章の1問目をコンパイル
gcc -Wall chapter01/exercise-1-1.c -o a.out

# 実行
./a.out
````
